# Notes App

Educational SPA application for taking and managing notes.

The project is created for learning and practicing React, TypeScript and basic frontend architecture.

---

## Tech stack

- React
- TypeScript
- Vite
- ESLint

---

## Features

- Simulated authentication
- Notes management
- Data fetching from a fake backend

---

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
